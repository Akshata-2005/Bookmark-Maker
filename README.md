# Bookmark Maker

This project is a simple **Bookmark Maker** web application built using **HTML, CSS, and JavaScript**. It allows users to save website names and URLs, and navigate to the bookmarked sites easily.

## Features
- Add a bookmark by entering a **site name** and **site URL**
- Form validation with error messages for empty inputs
- Each bookmark appears in a **list** with a clickable link
- Simple, clean user interface

## HTML Elements Used
- `<form>` with id `bookmarkForm`
- `<input>` elements with ids `siteNameInput` and `siteUrlInput`
- `<button>` element with id `submitBtn`
- `<label>` elements for inputs
- `<p>` elements with ids `siteNameErrMsg` and `siteUrlErrMsg` for error messages
- `<ul>` element with id `bookmarksList` to display the list of bookmarks
- `<a>` elements inside each bookmark item to navigate to the site

## How to Use
1. Open the `index.html` file in a browser
2. Enter a site name and URL in the form
3. Click the **Submit** button
4. See your bookmarks appear in the list, and click any link to visit the site
